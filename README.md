# Superstore Sales Dashboard

## Project Overview
This project is an interactive Power BI dashboard built using the Superstore dataset. It provides insights into sales performance, customer behavior, product performance, and regional trends.

## Features
- KPI Cards (Sales, Orders, Customers, Average Order Value)
- Monthly Sales Trend Analysis
- Regional Sales Analysis
- Category & Sub-Category Analysis
- Top Products by Sales
- Sales Target Tracking
- Interactive Filters and Slicers

## Tools Used
- Power BI
- DAX
- Data Visualization
